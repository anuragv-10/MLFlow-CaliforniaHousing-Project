# MLflow California Housing Project

This project demonstrates the use of MLflow for experiment tracking and model management using the California Housing dataset.

## Project Overview

The project focuses on:
- Training a Machine Learning regression model
- Tracking experiments using MLflow
- Logging parameters and metrics
- Model management and reproducibility

The California Housing dataset is used to predict housing prices based on different input features.

---

## Technologies Used

- Python
- MLflow
- Scikit-learn
- Pandas
- NumPy

---

## Dataset

Dataset Used:
- California Housing Dataset from Scikit-learn

Problem Type:
- Regression Problem

Target:
- Predict median house value

---

## MLflow Features Used

- Experiment Tracking
- Parameter Logging
- Metric Logging
- Model Logging
- MLflow UI Tracking

---

## Workflow

1. Load Dataset
2. Split Data into Training and Testing
3. Train Regression Model
4. Predict Output
5. Evaluate Model Performance
6. Log Experiments using MLflow
7. Store Metrics and Parameters
